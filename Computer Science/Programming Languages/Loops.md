#### For Loop
```C
for (initialization; condition; update) {
    // Code to be repeated
}
```

#### For Each Loop
```C
for (type element : collection) {
    // Code to be repeated
}
```

#### While Loop
```C
while (condition) {
    // Code to be repeated
}
```

#### Do While Loop
```C
do {
    // Code to be repeated
} while (condition);
```
